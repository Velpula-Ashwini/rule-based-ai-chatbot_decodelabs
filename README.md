# Rule-Based AI Chatbot

## Project Overview

This project is a Rule-Based AI Chatbot developed using Python and Flask. The chatbot interacts with users through a web interface and provides responses based on predefined rules and keyword matching.

## Features

* User-friendly chat interface
* Rule-based response generation
* Real-time interaction
* Fast and lightweight chatbot
* Easy to customize responses

## Technologies Used

* Python
* Flask
* HTML
* CSS
* JavaScript

## Installation

1. Clone the repository

```bash
git clone <repository-url>
```

2. Navigate to the project folder

```bash
cd chatbotdecodelab
```

3. Install dependencies

```bash
pip install flask
```

## Running the Project

Run the following command:

```bash
python app.py
```

Open your browser and visit:

```text
http://127.0.0.1:5000
```

## How It Works

The chatbot receives user input and checks it against predefined keywords. When a matching keyword is found, the corresponding response is displayed to the user.
